# 📸 Images Setup Guide

## Required Images

Save the following images in the **project root directory** (same location as README.md):

### 1. `architecture.png`
- System Architecture diagram 
- Location: `architecture.png` (project root)

### 2. `workflow.png`
- System Workflow diagram
- Location: `workflow.png` (project root)

---

## 📥 Quick Setup

1. Save `architecture.png` to project root
2. Save `workflow.png` to project root
3. Run these commands:

```bash
git add architecture.png workflow.png
git commit -m "Add system architecture and workflow diagrams"
git push origin main
```

That's it! Images will display in GitHub README.
